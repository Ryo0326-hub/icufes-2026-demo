# ICU祭 2026 デモサイト

ICU祭 2026 向けのデモWebサイトです。来場者がキャンパスマップ、出店情報、企画スケジュール、公式グッズ、アクセス情報、お知らせを確認できるようにした、フロントエンド中心のMVPです。

## 主な機能

- `ショップマップ`: キャンパスマップ上のエリアを選択し、該当エリアの出店を確認できます。
- `店舗検索`: 店名検索から対応する店舗番号へズームし、詳細を表示できます。
- `人気ランキング`: デモ投票数をもとに人気店舗トップ5を表示します。
- `企画スケジュール`: 10月11日・12日の企画を時間帯ごとに確認できます。
- `公式グッズ`: Tシャツ、トートバッグ、ステッカーなどのデモ商品を掲載しています。
- `アクセス・お知らせ`: 来場案内や最新情報を確認できます。

## 技術スタック

- Next.js App Router
- React
- TypeScript
- Tailwind CSS
- ESLint

## セットアップ

```bash
npm install
```

## 開発サーバー

```bash
npm run dev
```

ブラウザで `http://localhost:3000` を開いて確認します。

## 動作確認

```bash
npm run lint
npm run build
```

`npm run lint` でコード品質を確認し、`npm run build` で本番ビルドが通るか確認します。

## ディレクトリ構成

```text
src/app/            ページとルーティング
src/components/     UI・レイアウト・マップ・店舗・企画関連コンポーネント
src/data/           デモ用の静的データ
src/types/          TypeScript型定義
src/lib/            共通ユーティリティ
public/images/      背景画像、マップ画像、マスコット、プレースホルダー画像
```

## デモデータについて

現在は `src/data/` 配下の静的データを使用しています。実運用時には、同じデータ構造を保ちながらCMSやデータベースに差し替えることを想定しています。

## デプロイ

Next.js に対応したホスティング環境でデプロイできます。Vercelを使う場合は、GitHubリポジトリを連携し、ビルドコマンドに `npm run build` を指定してください。
